To run the files on the localhost,

Step1: Install node-js command prompt.
step2: Install mysql and MySQL Workbench 6.2 CE or latest on your system
Step3: import the dump file to microsoft sql work bench by clicking the server option and going to data import  option and select the file.
Step4:please connect to the mySql work Bench and import the dump file into your system.
step5: Go to that particular location or directory on your system and give node app.js command.
Step6: Go to the browser and give localhost:1234 and the Taxi web application will run on local environment.
